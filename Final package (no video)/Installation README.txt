1. Download and Extract the Zip File
    Extract the contents of the zip file into a desired folder on your computer.

2. Install Python
    Ensure Python (version 3.x or higher) is installed. If it’s not already installed, download it from Python.org.

3. Install Required Dependencies
    Open a terminal or command prompt and navigate to the folder where you extracted the zip file:
        cd /path/to/extracted/folder


4. Install Python Dependencies
    Run the following command to install the required Python packages:
        pip install -r requirements.txt


5. Run the Application
    To launch the application, run the following command:
        python multiscreen.py

Notes:
If Python isn’t installed or you run into issues with the installation, make sure you’re using Python 3.x. You can check the installed version by running:
    python --version
